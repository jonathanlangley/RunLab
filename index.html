RunLab

AI-powered running analytics for endurance athletes, quantified-self fans, and data nerds.

🚀 What is RunLab?

RunLab is a modular, Jupyter-based project that helps runners analyze their training using data science tools and AI.

📊 Features

  • Weekly mileage tracking vs. plan
  • Heart rate zone analysis
  • VDOT / fitness trend estimation
  • AI-generated summaries using OpenAI GPT
  • Exportable dashboards
